DESCRIPTION:

This project presents an interactive Brazil E-Commerce Sales Performance Dashboard built using the Brazilian e-commerce dataset. The dashboard analyzes sales performance, customer behavior, and order trends from 2016 to 2018 in a clear and data-driven way.

It highlights key business metrics such as Total Revenue, Total Orders, Total Customers, and Average Order Value. The dashboard also includes visualizations for monthly revenue trends, top 5 products by revenue, order distribution by state, payment method distribution, and revenue by product category. Interactive filters such as order date, product category, customer state, and payment type allow dynamic data exploration.

The project demonstrates how business intelligence and data visualization techniques can transform raw e-commerce data into meaningful insights that support better decision-making.
